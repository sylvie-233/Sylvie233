# 供应链


## 基础介绍

供应链（Supply Chain） 是指产品从最初的原材料采购开始，经过加工、制造、仓储、运输、销售，最终到达消费者手中的整个网络体系

核心流程：供应商 → 制造商 → 仓储与物流 → 分销商 / 批发商 → 零售商 → 客户




## 仓储

仓储是对货物在供应链中的存储、保管、调度与管理的活动

仓储业务流程：
1. 收货：收货检验、上架登记
2. 上架：指定货位、更新系统
3. 存储：定期盘点、保管养护
4. 拣货 / 配货：根据订单挑选货品
5. 发货：打包、出库扫描、装车

仓储相关职务：
- 仓储经理：规划、优化仓储流程
- 仓库主管：管理作业流程、调度人员
- 班组长：带领一线操作员执行任务
- 仓库管理员：收发货、入库、盘点
    - 收发货员：
    - 叉车司机：
    - 系统操作员：WMS 系统维护与数据录入

仓储信息化系统：
- WMS：仓储信息化系统
    - 入库管理：收货、质检、上架
    - 出库管理：拣货、复核、发运
    - 库存管理：实时库存、盘点、报警
    - 货位管理：自动分配货位
    - 作业调度：波次计划、任务分派
    - 接口系统：与 ERP、TMS 对接
    - 移动作业：PDA / 扫描枪 / RFID
- ERP：企业资源计划
- TMS：运输管理系统
- MES：制造执行系统
- SCM：供应链管理系统
