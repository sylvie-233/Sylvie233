# 公司法


## 基础介绍


公司按法律分类：
- 有限责任公司（有限公司 / Ltd.）：股东以出资额为限承担责任；常见于中小企业
- 股份有限公司（股份公司 / Co., Ltd.）：资本划分为股份，可以上市；股东以所持股份为限承担责任
- 个人独资企业：由一个自然人出资，个人承担无限责任
- 合伙企业：两个或以上自然人、法人共同出资；可为普通合伙或有限合伙
- 外商独资企业（WFOE）：外国法人或自然人独资设立
- 中外合资经营企业 / 合作企业：外国与中国企业或个人合资
- 国有独资公司：国家单独出资设立

公司按经营范围分类：
- 生产型企业：制造业、加工厂
- 贸易型企业：批发、零售、进出口贸易公司
- 服务型企业：咨询、IT服务、教育、金融、设计
- 科技型企业：软件开发、人工智能、生物科技
- 房地产企业：开发、租赁、物业管理
- 金融类企业：银行、保险、证券公司
